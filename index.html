Descripción del Proyecto:

Objetivos del Proyecto:

  • Usar buenas normas de programación para llevar a cabo una correcta estructuración de la aplicación de asistencias.
  • Visualizar e implementar interfaces amigables para mejorar la experiencia del usuario.

Información del proyecto:

  • Con este proyecto espero realizar un buen sistema de asistencia mediante el lenguaje de programacón Java usando el siempre confiable NetBeans IDE y tambien un poco de html. Su funcionamiento es el siguiente, el docente y estudiante podran ingresar cada uno a su respectiva interfaz en donde el docente podra registrar sus datos y la materia que imparte y el estudiante podra registrar sus asistencias con un si.
